============
Installation
============

The recommended way to install slewpy is using `pip <https://pip.pypa.io/en/stable/>`_

.. code-block:: console

    pip install slewpy

You can also install slewpy via conda-forge

To install from source

.. code-block:: console

    git clone https://github.com/LLNL/slewpy.git
    cd slewpy
    pip install .
